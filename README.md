## Backchat

#### Miniature chat app created in Backbone JS

Visit [Backchat](https://backchat62.herokuapp.com/) to use.

To download:

- ```git clone https://github.com/dwatson62/backchat```
- ```cd backchat```
- ```npm start```
- Visit [Localhost](http://localhost:3000/)
